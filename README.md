# Diplom_VK_bot
## Схема данных favorites.json
Файл содержит список избранных пользователей. Каждый пользователь — это объект (словарь) со следующими полями:

*`vk_id` (int): Уникальный идентификатор пользователя ВКонтакте.

*`first_name` (str): Имя пользователя.

*`last_name` (str): Фамилия пользователя.

*`profile_url` (str): Ссылка на страницу пользователя ВКонтакте. Формируется как `https://vk.com/id{vk_id}`.

*`photos` (array of strings): Три самых популярных фотографии в виде attachment строк для ВКонтакте**. Каждая строка имеет формат `photo{owner_id}_{photo_id}`, который используется для отправки вложений в сообщениях

*`saved_at` (str): Дата и время добавления пользователя в избранное в формате `ГГГГ-ММ-ДДTЧЧ:ММ:СС` (например, `"2025-04-05T14:30:00"`).
