# Diplom_VK_bot
## Описание
Бот для знакомств ВКонтакте, который ищет людей по заданным критериям и сохраняет избранных в JSON-файл.
### Схема данных favorites.json
Файл содержит список избранных пользователей. Каждый пользователь — это объект (словарь) со следующими полями:

* `vk_id` (int): Уникальный идентификатор пользователя ВКонтакте.

* `first_name` (str): Имя пользователя.

* `last_name` (str): Фамилия пользователя.

* `profile_url` (str): Ссылка на страницу пользователя ВКонтакте. Формируется как `https://vk.com/id{vk_id}`.

* `photos` (array of strings): Три самых популярных фотографии в виде attachment строк для ВКонтакте**. Каждая строка имеет формат `photo{owner_id}_{photo_id}`, который используется для отправки вложений в сообщениях

* `saved_at` (str): Дата и время добавления пользователя в избранное в формате `ГГГГ-ММ-ДДTЧЧ:ММ:СС` (например, `"2025-04-05T14:30:00"`).


### Настройка
1. Установите зависимости: `pip install vk-api`
2. Скопируйте конфиг: `cp config.example.py config.py`
3. Заполните `config.py` своими токенами:
   - `VK_GROUP_TOKEN` — токен группы ВК
   - `VK_USER_TOKEN` — токен пользователя ВК (получить на vkhost.github.io)
4. Укажите ID вашей группы в `GROUP_ID`

## Структура проекта
- `main.py` — основной файл бота
- `vk_tools.py` — модуль работы с VK API
- `data_handler.py` — модуль работы с JSON-файлами
- `config.py` — настройки (не коммитится)
- `favorites.json` — данные избранных пользователей

## Функциональность
- Поиск пользователей по возрасту, полу и городу
- Получение 3 самых популярных фотографий
- Сохранение пользователей в избранное
- Просмотр списка избранных
